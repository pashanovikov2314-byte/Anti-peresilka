Как это работает

Система использует Telegram Bot API для мониторинга уведомлений о скриншотах и пересылках. При обнаружении подозрительной активности автоматически отправляются оповещения админам.

  • ✅ Отслеживание системных уведомлений
  • ✅ Анализ пересылок между чатами
  • ✅ Рейтинг пользователей по риску
  • ✅ Экспорт данных в CSV/JSON
